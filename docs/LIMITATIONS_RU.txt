# Limitations

Kit B v1 is a real backend/FBO/depth-pass prototype, but still not final.

What it does:
- adds cubemap face attachment
- clears depth
- runs drawSurf list in depthFill mode
- uses selected dlight cubemap in shade path

What it may still lack:
- true point-light cubemap face matrices
- correct far/near for each dlight
- depth compare/bias integration if lightall shader expects another format
- PCF
- robust selection by semantic light type

Next step after testing:
Kit C: true cubemap face view matrices + shader depth compare adjustment.
