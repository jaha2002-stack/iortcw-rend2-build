# Limitations

This kit tries to implement real light-face matrices, but it may still need one more stage.

What Kit C does:
- selected dlight
- six cubemap faces
- temporary light origin view
- 90-degree projection
- Mat4View-based modelview
- restores player view

What may still be missing:
- exact handedness/axis mapping for RTCW/rend2 cubemaps
- correct depth format/linearization
- correct shader-side distance-vs-depth compare
- bias actually passed into GLSL
- PCF/filtering

If Kit C logs CUBEVIEWDBG for faces 0-5 but shadows are still not visible,
next kit should be Kit D: shader depth compare + bias debug.
