# Practical testing notes

1. Проверить cvars:
   /r_dlightMode
   /r_dlightShadows
   /r_dlightShadowMaxLights
   /r_dlightShadowMapSize
   /r_dlightShadowBias
   /r_dlightShadowFilter
   /r_dlightShadowDebug
   /r_dlightShadowIntensity

2. Тест факелов:
   /exec rend2_mode3_shadowmap_torch.cfg

3. Тест двух источников:
   /exec rend2_mode3_shadowmap_two_lights.cfg

4. Тест солнца:
   /exec rend2_sun_shadow_test.cfg

5. Тест взрывов/пулемёта:
   /exec rend2_mode3_action_lights_test.cfg

Совет:
Держать /r_specularMapping 0 при тесте теней, иначе мокрый блеск пола маскирует эффект.
