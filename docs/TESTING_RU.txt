Use the same torch location.

Run each cfg and compare visually:
1. /exec rend2_kit_e_1_force_shadow.cfg
2. /exec rend2_kit_e_2_distance_debug.cfg
3. /exec rend2_kit_e_3_cubemap_sample_debug.cfg
4. /exec rend2_kit_e_4_actual_nonlin_compare.cfg
5. /exec rend2_kit_e_5_actual_linear_compare.cfg
6. /exec rend2_kit_e_6_inverted_nonlin_compare.cfg
7. /exec rend2_kit_e_7_direction_pattern.cfg

Then:
/exec rend2_kit_e_verbose_logs.cfg
/condump kit_e_result.txt

Interpretation:
- Mode 1 no visible change: shader path/uniform path broken.
- Mode 2 no visible change: var_WorldPos or u_LightOrigin problem.
- Mode 3 no visible change: linear-distance cubemap is not being filled or sampled.
- Mode 3 changes but mode 4/5 no shadows: bias/compare/face axes issue.
- Mode 4/5 shadows appear: Kit E succeeds.
