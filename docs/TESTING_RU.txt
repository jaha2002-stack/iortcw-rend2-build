# Testing Kit C

1. Build in GitHub Actions.
2. Install artifact.
3. In game:
   /exec rend2_kit_c_lightview_first.cfg

4. Open console. Look for:
   CUBEVIEWDBG light=0 face=0 rendered=1
   CUBEVIEWDBG light=0 face=1 rendered=1
   CUBEVIEWDBG light=0 face=2 rendered=1
   CUBEVIEWDBG light=0 face=3 rendered=1
   CUBEVIEWDBG light=0 face=4 rendered=1
   CUBEVIEWDBG light=0 face=5 rendered=1

5. If no visible shadows:
   /exec rend2_kit_c_lightview_verbose.cfg
   /condump kit_c_lightview.txt

Send screenshot + 20 lines from kit_c_lightview.txt.

6. If game crashes or screen goes wrong:
   /exec rend2_kit_c_compare_currentview.cfg
This disables new light-view matrices and reverts to Kit B-style behavior.
