# Testing

1. Соберите через GitHub Actions.
2. Установите artifact.
3. В игре:
   /exec rend2_kit_b_depthpass_first.cfg

4. Откройте консоль.
5. Ищите:
   CUBEPASSDBG light=0 face=0 rendered=1
   CUBEPASSDBG light=0 face=1 rendered=1
   ...
   CUBEPASSDBG light=0 face=5 rendered=1

Если CUBEPASSDBG есть, значит backend depth-pass реально выполняется.

Если CUBEPASSDBG есть, но теней нет:
- вероятно, cubemap не depth-capable
- или shader compare не использует depth как нужно
- или нужен Kit C с real light-face matrices

Если сборка упадет:
пришлите последние 30-50 строк ошибки.
