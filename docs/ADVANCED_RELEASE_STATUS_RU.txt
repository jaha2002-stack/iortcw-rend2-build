# Advanced renderer release: честный статус

Запрошенные функции:

1. Тени от факелов
2. PBR dynamic light
3. Specular highlight от каждого dynamic light
4. Реакция normal maps на факелы / dlights

Честно: это НЕ готово как быстрый патч.

Причина:
текущий r_dlightMode 1 в rend2 - это отдельный projected dynamic-light pass.
Он не имеет полноценного доступа к material normal/specular maps так, как основной material pass.
Чтобы сделать advanced release правильно, нужно менять C-код renderer backend, material binding и GLSL.

## Что нужно сделать для настоящего Advanced Release

### Этап A: normal/specular dynamic light без теней

1. В backend renderer найти dlight pass.
2. Передавать в dlight shader:
   - normal map текущего материала
   - specular map текущего материала
   - tangent/binormal или TBN matrix
   - view vector
   - specularReflectance
   - specularExponent
3. В dlight fragment shader считать:
   - normal from normal map
   - diffuse = max(dot(N,L),0)
   - specular через half-vector
   - attenuation
4. Добавить fallback для материалов без normal/specular maps.

Это реально и должно быть следующим большим патчем.

### Этап B: тени от факелов

Для point light shadows нужны:
- выбор ближайших dynamic lights
- shadow cubemap или dual-paraboloid shadows
- depth pass для каждого light
- bias
- PCF filtering
- лимиты качества
- отключение для слабых/далёких lights

Это большой renderer project. Его нельзя честно выдать как рабочий релиз без тестирования.

## Реальный порядок работ

1. Release 1: enhanced r_dlightMode 1 shader-only patch. ГОТОВО в этом builder kit.
2. Release 2A: dlight normal/specular material support. Требует C-кода renderer.
3. Release 2B: point-light dynamic shadows. Большая отдельная задача.
