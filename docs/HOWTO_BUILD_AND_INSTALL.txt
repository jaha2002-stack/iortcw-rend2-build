How to build
1. Create a new empty GitHub repository.
2. Upload all files from this kit, including the hidden .github folder.
3. Open Actions.
4. Run: Build Rend2 Mode3 Torch Shadows Best Effort Windows x64
5. After completion, download artifact:
   iortcw-rend2-mode3-torch-shadows-best-effort-windows-x64

How to install
1. Unzip the artifact.
2. Copy new .exe and .dll files into your RTCW/iortcw folder.
3. Copy cfg files into RTCW/main/.
4. Start the game.
5. In console run:
   /exec rend2_mode3_torch_shadows.cfg
