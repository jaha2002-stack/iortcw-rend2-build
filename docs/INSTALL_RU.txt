# Установка собранного релиза iortcw Rend2 Enhanced

Этот комплект НЕ содержит оригинальные игровые данные RTCW.
Он собирает только open-source engine files iortcw.

## Как собрать без программирования

1. Создайте новый пустой репозиторий на GitHub.
2. Загрузите туда ВСЕ файлы из этой папки:
   - .github/workflows/build-rend2-enhanced-windows-x64.yml
   - patches/iortcw-rend2-dlightmode1-enhanced-perpixel.patch
   - scripts/rend2_dlightmode1_enhanced_test.cfg
   - docs/*.txt
3. Откройте вкладку Actions.
4. Выберите workflow: Build iortcw Rend2 Enhanced Windows x64.
5. Нажмите Run workflow.
6. Дождитесь окончания сборки.
7. Внизу страницы сборки скачайте Artifact:
   iortcw-rend2-enhanced-dlight-windows-x64

## Как установить в игру

1. Распакуйте artifact.
2. Скопируйте собранные .exe и .dll в папку iortcw.
3. Оригинальные RTCW pak-файлы должны находиться в iortcw/main/.
4. Скопируйте rend2_dlightmode1_enhanced_test.cfg в iortcw/main/.
5. Запустите ioWolfSP.x64.exe или ioWolfMP.x64.exe.
6. В консоли выполните:

   /exec rend2_dlightmode1_enhanced_test.cfg

## Что улучшено в Release 1

- r_dlightMode 1 получает более мягкий per-pixel falloff.
- Убрана жёсткая граница step() по normal.
- Добавлено radial attenuation.
- Уменьшен projected texture leak.
- Добавлен clamp против грязного пересвета.

## Чего нет в Release 1

- Нет настоящих dynamic shadows.
- Нет normal/specular reaction от каждого dynamic light.
- Нет PBR dynamic lights.

Это честный безопасный патч, который не ломает основной renderer pipeline.
